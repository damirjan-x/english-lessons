# Шаблон отчёта

Положите в эту папку файл **report_template.docx** — шаблон отчёта в формате Word.

## Плейсхолдеры для подстановки (Jinja2 / docxtpl)

В тексте шаблона используйте переменные в **двойных квадратных скобках** `[[ имя_переменной ]]` (не `{{ }}`, чтобы случайные фигурные скобки в Word не ломали шаблон):

| Переменная    | Описание                    |
|---------------|-----------------------------|
| `[[ first_name ]]` | Имя студента (рус.)       |
| `[[ last_name ]]`  | Фамилия студента (рус.)   |
| `[[ first_name_he ]]` | Имя на иврите          |
| `[[ last_name_he ]]`  | Фамилия на иврите       |
| `[[ year ]]`       | Год                       |
| `[[ month ]]`      | Месяц (число 1–12)        |
| `[[ month_ru ]]`   | Название месяца на русском (январь, февраль, …) |
| `[[ month_he ]]`   | Название месяца на иврите (ינואר, פברואר, …) |
| `[[ GrammarE ]]`   | Grammar (English), до 255 |
| `[[ ReadingE ]]`   | Reading (English)         |
| `[[ SpeakingE ]]`  | Speaking (English)        |
| `[[ WritingE ]]`   | Writing (English)         |
| `[[ hours_studied ]]` | Количество часов (1–12) |

Пример фрагмента в .docx:  
«Отчёт по студенту [[ first_name ]] [[ last_name ]] за [[ month_ru ]] [[ year ]]. Grammar (E): [[ GrammarE ]].»
